# OTVerb
OTVerb is a dirty Reverb VST plugin.
I made with cycling Max(gen~) + JUCE(C++).

I aim to make sound for some janre such as shoegaze, dreampop, idm, ambient.

The OTVerb is implemented from Max for live devices.
